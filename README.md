# Wendy and the Tiny Helper
*A children's-book-style introduction to building AI agents in Claude.*

## Who this is for
Non-technical professionals — creatives, lawyers, operators — who feel like "agents" are complicated, mysterious, and not for them. The story format does three things:
1. Lowers the stakes (it's a picture book — nobody's expected to "get" it in one read).
2. Anchors the abstractions to a concrete, relatable use case.
3. Gives them a vocabulary they can carry into a real conversation about building one.

## Use case
The protagonist is **Wendy, a General Counsel** drowning in vendor contracts. Each chapter teaches a layer of agent design through her story.

## Chapters

| # | Title | Concepts introduced |
|---|-------|--------------------|
| 1 | Wendy and the Tiny Helper | What an agent is; the loop (read → think → act); system prompt; tools; evals; guardrails |
| 2 | The Helper Gets Helpers | Sub-agents; specialization; delegation; why narrow scope works better than one super-helper |
| 3 | How Wendy Built Her Helper | The actual setup in Claude Desktop using Cowork — role-matched plugins, connecting tools, running a skill via `/` commands |

Future chapters could cover: memory, human-in-the-loop, when *not* to use an agent, evaluating an agent's work over time.

## Format
Each chapter is a markdown file. Each numbered beat is one spread of the book. Under each beat is an **Illustration** brief describing what the picture should show. The text is meant to be read aloud — short sentences, lots of white space.

## Visual direction (suggested)
- **Style**: Warm, hand-drawn, slightly whimsical. Think modern picture book — not corporate.
- **Palette**: Cream paper, navy and warm gold for the "legal" world; the agent glows soft amber-teal.
- **Wendy**: A relatable adult, mid-career, not a caricature. She's the audience surrogate.
- **The agent**: A small, round, glowing creature with big friendly eyes. Not a robot. Not a person. Something in between — a sprite. This matters: avoiding robot iconography keeps the metaphor closer to "thoughtful helper" than "machine."
- **The office**: Real, but with one whimsical touch per page (the contract pile bumping the ceiling, papers arranged in a polite line, the agent perched on a coffee mug).

## How to revise
This is a first draft. Open suggestions:
- Tighten or expand any beat — current Chapter 1 is ~12 spreads, which is on the long end for a picture book.
- Try a different protagonist for a sister edition (a marketing creative, an HR lead). Same structure works.
- Rhyme pass? The current draft is rhythmic prose. Some beats could rhyme; full rhyme is risky to do well.
